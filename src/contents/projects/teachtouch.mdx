## Background

<img 
  src="/projects/teachtouch/tt-background.png" 
  alt="Background" 
  className="mx-auto rounded-lg w-196 my-4" 
/>
키오스크 도입률은 매년 상승하고 있지만 고령층의 활용 가능성은 단 17.9% 밖에 되지 않습니다. 더불어 수요는 매년 증가함에도 디지털 교육에 대한 지원과 예산은 줄어들고 있습니다.

따라서 이러한 상황 속에서 디지털 배움터와 같은 오프라인 교육 시스템의 한계를 극복하기 위해 다음 두 가지의 핵심 기능을 잡고 개발을 시작했습니다.
- 시간·공간적으로 제약이 있는 오프라인 교육보단, 본인의 디지털 기기로 즉시 학습이 가능하도록
- 일회성 교육으로 이해해야 하는 배움보단, 수준에 맞게 필요한 교육을 반복 학습

## What I Did

- 6인 팀에서 TypeScript와 React를 사용해 프론트엔드를 설계·개발
- 로그인/회원가입, 키오스크 구성 학습, 주문 및 결제하기 학습, 홈, OCR ui 개발 
- 학습 가이드 검색 및 상태 저장, tts, ocr, 기본 user api 연결
- framer-motion을 사용해 자연스러운 상호작용을 제공
- AWS S3를 사용해 배포

<br/>
## Result

TeachTouch는 다음 네 가지 장점을 갖고 있습니다.
- 실제 키오스크 사용에 대한 두려움 해소
- 반복 학습을 통한 사용 자신감 향상
- 디지털 약자의 사회적 자립 지원
- 교육자/보호자가 함께 활용할 수 있는 보조 학습 도구로서 가치
