## Background

포스터나 카드뉴스를 제작하는 업무를 맡던 중 전임자가 만들어둔 작업 틀을 그대로 유지하면서 내용을 수정해야 하는 경우임에도, jpg로만 저장되어있어 간단한 수정이 불가능해 처음부터 다시 만들었던 실제 어려움에서 출발했습니다.

이미지를 템플릿 형태로 재구성하여 배경 색상을 바꾸거나, 객체를 재배치하거나, 텍스트를 바꾸는 등의 편집을 손쉽게 할 수 있는 도구가 있다면 훨씬 효율적이겠다는 개인적인 필요에 의해 시작한 프로젝트입니다.

<br/>
## Key Features

- 객체 자동 분리 (Meta의 Segment Anything Model 기반)
- 객체 제거 후 배경 자동 복원 (inpainting)
- 개별 객체 편집 (위치 이동, 색상 변경, 삭제 등)
- 브러시 및 도형을 활용한 객체 확장
- 레이어 기반 저장 및 템플릿 재사용
<br/>

## What I Did

- 활용할 모델 리서치
- 객체 분리를 위해 오픈소스 편집 툴인 napari 위에 Meta가 공개한 Segment Anything Model(SAM) 모델 결합
- 배경 복원 과정에서 이미지 생성 모델인 LaMa 모델 결합
- 레이어 기반 개별 객체 export 기능
